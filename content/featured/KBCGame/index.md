 Developed a Kaun Banega Crorepati (KBC) game in C, where players answer multiple-choice questions to win virtual prizes. The game follows the classic KBC format, featuring a question ladder and lifelines to assist players. It includes a user-friendly console interface, randomized questions, and a scoring system to track progress. Designed for both fun and learning, this project showcases my C programming skills, including file handling, loops, and conditional logic.
